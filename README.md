# 俗世神人

一个本地运行的网络争议人物记录网站，用来整理褒贬不一、评价两极、在互联网上留下强烈公众印象的人物。

## 功能

- 手动录入人物记录
- TXT / JSON 导入
- 编辑、删除、导出
- 本地搜索
- 审核状态：待审核、已通过、已驳回
- AI 功能已暂时封存，保留为“有待开发”

## 启动

确保本机已安装 Node.js，然后在项目目录运行：

```bash
node server.js
```

打开：

```text
http://localhost:4173
```

也可以直接打开 `index.html` 使用纯静态模式；此时记录会保存在浏览器本地。

## 数据

通过本地服务运行时，记录会保存在 `data/people.json`。  
点击页面里的“导出 JSON”可以做备份。

## 投稿与审核

GitHub Pages 是静态网站，不能直接把访客提交的内容写入站主电脑或仓库。  
访客在网页中保存的记录会留在访客自己的浏览器里。

推荐流程：

1. 朋友在网页里填写记录。
2. 朋友点击“导出 JSON”。
3. 朋友把 JSON 文件发给站主。
4. 站主在自己的页面导入 JSON。
5. 站主在“待审核”里选择通过或驳回。

## 项目结构

- `index.html` 页面结构
- `styles.css` 页面样式
- `app.js` 前端逻辑
- `server.js` 本地服务

## 部署

仓库包含 GitHub Pages 工作流，公开仓库并启用 Pages 后会自动部署静态页面。
